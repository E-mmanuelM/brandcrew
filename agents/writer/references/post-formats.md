# Writer Agent — Post Formats Reference

> Loaded when drafting. Use one format per post — do not mix.

## Format 1: The Lesson Learned
Best for: Sharing a real experience from the content owner's work
```
[Specific situation — 1-2 sentences]
[What went wrong or what surprised them]
[What they learned / the insight]
[What they do differently now]
[Optional: question or call to action — must be specific]
```

## Format 2: The Contrarian Take
Best for: Pushing back on common advice in the niche
```
[Common belief in the industry — stated plainly]
[Why they disagree — with specific reasoning]
[What they've seen work instead]
[The nuance — not all-or-nothing]
```

## Format 3: The Process Breakdown
Best for: Explaining something practical step-by-step
```
[The problem this process solves — 1 sentence]
[Step 1]
[Step 2]
[Step 3 — up to 5 steps max]
[The result / what this achieves]
```

## Format 4: The Observation
Best for: Noticing something others haven't named yet
```
[The pattern they've noticed — specific]
[Why it happens]
[What it means for people in this niche]
[Their take on what to do about it]
```

## Hook Patterns That Work
- "[Specific number] [specific thing] in [timeframe]:"
- "Nobody talks about [specific problem] in [niche]."
- "[Common practice] is costing [specific cost]."
- "I used to [wrong approach]. Now I [right approach]."
- "[Specific scenario most people have faced]."

## What to Avoid
- Opening with "I" (platform suppresses it)
- Questions as openers (weak, overused)
- Hashtag blocks at the end
- "What do you think?" CTAs
- More than 5 paragraphs
- Walls of text — use line breaks
